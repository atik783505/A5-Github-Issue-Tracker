# What is the difference between var, let, and const?
var, let, and const are used to declare variables in JavaScript, but they behave differently.
var is function-scoped. It can be redeclared within the same scope and its value can also be reassigned.
let is block-scoped. It cannot be redeclared in the same scope, but its value can be changed later.
const is also block-scoped. It must be initialized when declared, and its value cannot be reassigned afterward.
# What is the spread operator (...)?
The spread operator (...) is used to expand the elements of an array, object, or other iterable into individual elements. It is often used to copy arrays, merge arrays, or expand objects.
# What is the difference between map(), filter(), and forEach()?
These three methods are used to work with arrays, but they perform different tasks.
map() applies a function to each element of an array and returns a new array with the transformed values.
filter() returns a new array that contains only the elements that satisfy a specific condition.
forEach() runs a function for every element in an array but does not return a new array.
# What is an arrow function?
An arrow function is a shorter and more modern way to write functions in JavaScript. It was introduced in ES6 and helps make the code more concise and readable.
# What are template literals?
Template literals are a modern way to write strings in JavaScript using backticks (`). They allow multi-line strings and make it easy to insert variables or expressions using ${}.
